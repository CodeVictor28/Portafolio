# Portafolio de Victor Ledesma Vega

Portafolio profesional desarrollado con React y Vite. Presenta mi perfil,
proyectos, experiencia laboral, formación, certificados y canales de contacto
mediante una interfaz moderna, animada y adaptable a diferentes dispositivos.

## Tecnologías principales

- React 19 y React Router
- Vite
- Motion, GSAP y OGL
- CSS responsivo
- Netlify

## Ejecutar el proyecto en Visual Studio Code

1. Abre la carpeta del proyecto en Visual Studio Code.
2. Abre la terminal integrada.
3. Instala las dependencias:

   ```bash
   npm install
   ```

4. Inicia el servidor local:

   ```bash
   npm run dev
   ```

## Verificación

```bash
npm run lint
npm run build
```

## Despliegue

El proyecto incluye la configuración necesaria para desplegarse en Netlify,
incluyendo soporte para las rutas internas de la aplicación.
